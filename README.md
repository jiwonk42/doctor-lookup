# _Doctor Lookup_

#### _3.17.2017_

#### By _**Jiwon Kang**_

## Description

_This application will allow the user to look up doctors who specialized in any medical issue entered by the user_

## Setup Instructions
* Clone this repository
```
$ npm install
$ bower install
```
* Open index.html with a web browser
```
$ gulp build
$ gulp serve
```
